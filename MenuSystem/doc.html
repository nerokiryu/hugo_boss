MenuSystem.init()

    initialise les globales DISPLAY et DISPLAYRECT
    doit-être appelé après initialisaton de la fenêtre pygame


class Menu:

    ils ne devraient normalement pas être utilisés tels quels même si celà est possible
    les menus sont destinés à être utilisés par les conteneurs suivants:
    -MenuSystem
    -MenuFix
    -MenuBar
    -Menuchoice
    le menu dont le contenu dépasse le cadre est scrollé via la molette de la souris

MenuSystem.Menu(self,label,itemslist,exc=()):

        -label de type str est le titre du menu, il sera utilisé si ce menu sert de sous-menu
        -itemslist est un iterable contenant des chaînes et/ou des menus
        -exc est une suite d'indexs d'entrées non utilisées(exlusion), ces entrées seront "grisées" et ne répondront pas au survol de la souris


class MenuSystem:

      c'est le conteneur de base
      le constructeur ne nécessite aucun argument

méthodes

MenuSystem.set(menu,pos,w=None,h=None,force_pos=False)

        initialise et affiche un menu; le menu disparaîtra au clic de souris, sauf si le clic se fait sur une entrée vers un sous-menu
        -menu est un Menu
        -pos est un couple de coordonnées
        -w et h sont la largeur et la hauteur du menu en pixel; donner une valeur force la dimension, laisser à none elle sera calculée
        -force_pos est un booléen qui force la position du menu, s'il vaut False le menu sera placé automatiquement

MenuSystem.update(pygame.event.Event)

        reçoit un événement est actualise le menu en conséquence

MenuSystem.clear()

        vide le MenuSystem des menus qu'il contient, ce qui a pour effet des les effacer de l'écran

MenuSystem.redraw()

        redessine le menu sans le réinitialiser si nécessaire

property

MenuSystem.choice

        renvoit le choix fait pas l'utilisateur
        c'est une liste de tuple (index,str)
        le setter ne devrait pas être utilisé

MenuSystem.select

        renvoit l'entrée pointée par la souris sous forme d'une liste de tuple (index,str)

attributs

MenuSystem.choice_index

        renvoit le choix fait pas l'utilisateur sous forme de liste d'index

MenuSystem.choice_label

        renvoit le choix fait pas l'utilisateur sous forme de liste de labels
